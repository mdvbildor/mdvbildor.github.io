Perguntas Frequentes

  • Como funciona a licença vitalícia?

    Acreditamos que a melhor forma de entregar nosso produto aos clientes é a licença vitalícia. Você não se torna dependente de planos de assinaturas mensais, compra o produto uma vez e pode usá-lo para sempre, além de poder receber atualizações.

  • Como configurar o Pexels no Bildor?

    Para configurar a integração com o Pexels, basta criar uma conta no site pexels.com e ir para a página de API. Leia atentamente as políticas de privacidade e uso da API e gere uma chave.
    A sua chave da API deve ser guardada com segurança e poderá ser usada no Bildor para buscar imagens automaticamente.

  • O Bildor possui limite?

    Após comprar o software você poderá gerar quantos posts quiser por mês de forma ilimitada.
    Caso esteja usando a API do Pexels para buscar imagens, no entanto, fique atento ao limite da plataforma.
    Fique atento aos limites do Pexels. Por padrão, a API tem uma taxa limitada a 200 solicitações por hora e 20.000 solicitações por mês.

  • O Bildor agenda minhas postagens automaticamente?

    Não. O Bildor é uma ferramenta que auxilia na criação de uma grande quantidade de imagens em pouco tempo. A publicação dessas imagens poderá ser feita manualmente no Instagram ou utilizando gerenciadores de redes socials como o mLabs ou later.com, por exemplo.
